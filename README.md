For this project I built a weather web app using HTML, CSS and JavaScript that lets the user enter the name of a US city 
and it outputs the tempaure of the city the user can also pick temparure they wanna use like Fahrenheit or Celsius. 
I used a weather api for the fetch the weather data.


The weather app provides user with data like:

-Current weather

-Weather description

-Emoji based on weather condition

-Max/High tempauture

-Humidity

>NOTE: You do have to intert your api key on  'YOUR_API_KEY' for the data to fetch. 
For this project I used an APIfrom OpenWeather https://openweathermap.org to fetch weather data.
You can get an API key for free by creating an account on OpenWeather.
